# Primary sources (Git LFS)

Original primary-source documents for the Raschpëtzer qanat model, stored via
**Git LFS** (see `/.gitattributes`). These back the citations in
`data/sources.json` (`waringo2018`, its figures, and the SIT leaflet).

| File | Description |
|------|-------------|
| `Raschpëtzer Brochure_en.pdf` | Faber/Waringo/Werner (2018), *The Raschpëtzer — A Roman Underground Water Supply System*, SIT Walferdange (ISBN 978‑2‑9199454‑2‑9). The primary reference. |
| `20170501 Dépliant Rachpëtzer_en.pdf` | SIT Walferdange visitor leaflet (EN). |
| `Fig_2-1a_Quell-Schuettungen.jpg` | Figure 2‑1a — spring discharge. |
